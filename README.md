 
# This is an Interactive Personal Data Collector

This project is used to collect and store the personal information of user.

This project is built to perform following tasks:

1.) I have taken personal data from user using input().
2.) I have checked the input data type using type().
3.) I have converted the necessary data types into different data types using float(), int() wherever it is necessary.
4.) I have checked the id or memory address of the input data using id().
5.) I have calculated the birth year of a user according to his/her age using subtraction operator().
6.) I have print all result using print().
