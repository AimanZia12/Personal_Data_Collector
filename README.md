 
# This is an Interactive Personal Data Collector

This is built to collect and store the personal information of user.

This project is built to perform following:

1.) First I have taken data input from user.
2.) I have checked the input data type using type().
3.) I have converted the necessary data types into different data types using float(), int().
4.) I have checked the id of the input data using id().
5.) I have calculated the birth year of a user according to his/her age.
